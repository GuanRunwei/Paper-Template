Note: 
1. For any inquiry about the template, please see detailed guidelines from following website: 
https://www.acm.org/publications/proceedings-template

2. Word file for submission is preferred.

3. Find your CCS Concepts (ACM Computing Classification System, which category your paper belongs to) and add it to your paper: 
https://dl.acm.org/ccs/ccs.cfm
Below is an example for reference (Please select the appropriate CCS Concepts level by level and add them to the paper, fill in at least 3 levels):
Computing methodologies-Artificial intelligence-Computer vision-Computer vision problems-Object detection

4. Full name of each author, no abbreviation. 
   Only one corresponding author and his/her name must be marked with a Asterisk*. 

5. Add email address of each author and make sure their email addresses are different.
